# Proyecto de Login y Dashboard de Subsidios de Seguros Agrícolas

Este proyecto tiene como objetivo desarrollar un sistema de login y un dashboard para gestionar los subsidios de seguros agrícolas. Los usuarios podrán acceder fácilmente a la información de sus subsidios, realizar consultas y gestionar su documentación relacionada.

## Características Principales
- Registro y autenticación de usuarios
- Visualización de subsidios disponibles
- Gestión de solicitudes y documentación
- Dashboard intuitivo para una mejor experiencia del usuario

Este proyecto busca facilitar el acceso a los subsidios de seguros agrícolas y proporcionar una herramienta útil para los beneficiarios.
